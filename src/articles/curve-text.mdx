# Curved Text Effects

SVG's `<textPath>` element allows you to bend text along any path, opening up creative possibilities for unique typography.

---

## The TextPath Element

The `<textPath>` element references a path and flows text along it. This is perfect for circular logos, wavy headlines, or any creative text layout.

## Positioning and Alignment

The `startOffset` attribute controls where text begins on the path. Combined with `textAnchor`, you can precisely position text anywhere along the curve.

### Common Patterns

- **startOffset="0%"** - Start at the beginning
- **startOffset="50%" + textAnchor="middle"** - Center the text
- **startOffset="100%" + textAnchor="end"** - End-aligned text
